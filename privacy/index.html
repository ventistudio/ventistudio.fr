Politique de confidentialité

Protection de vos données

Données collectées

VentiStudio ne conserve que les informations suivantes :

  • Nom d'utilisateur
  • Mot de passe (crypté)
  • Code à usage unique (généré lors de l'inscription)

Utilisation du code à usage unique

Le code à usage unique est généré lors de votre inscription et permet de :

  • Réinitialiser votre mot de passe en cas d'oubli
  • Prouver votre identité pour les demandes RGPD

Conservation des données

Pour des raisons techniques, nous ne pouvons pas supprimer automatiquement les comptes inactifs depuis plus de 3 ans. Cependant, nous supprimerons vos données sur demande dans les cas suivants :

  • Demande RGPD
  • Demande "Do not share my personal information"

Note : Votre code à usage unique sera nécessaire pour valider ces demandes.

Partage des données

VentiStudio ne partage aucune donnée personnelle avec des tiers. Cependant, certains services externes vers lesquels nous redirigeons peuvent avoir leurs propres politiques de collecte et de partage de données.
