Guide d'installation d'Aina Umaishi en Local

1. Prérequis

  • Un environnement Python 3.9 ou supérieur.
  • Un GPU compatible avec CUDA pour des performances optimales.

2. Choisir une Application

Télécharger Msty Télécharger Jan Télécharger Jellybox

3. Télécharger le Modèle LLM

  • Téléchargez le modèle recommandé depuis le site officiel ou Hugging Face.
  • Les configurations recommandées sont listées ci-dessous :
Configuration Modèle CPU GPU NPU RAM Stockage
Minimal Llama 3.2 3B Intel i5 7ème génération GTX 1060 ou équivalent Non requis 8 Go SSD 5 Go
Recommandé Llama 3.2 Vision 11B Intel i7 10ème génération RTX 3060 ou équivalent Non requis 16 Go SSD 10 Go
Optimal Llama 4 Scout AMD Ryzen 9 5900X RTX 4090 ou équivalent Compatible NPU 32 Go SSD 20 Go
